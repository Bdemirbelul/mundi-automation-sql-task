# Mundi – SQL Automation Task

This repository contains my SQL solution for the Mundi data task.

## Task
Calculate the rolling 3-day average of total automation runs per day and report the value for **March 15**.

## Approach
1. Aggregate automation runs per day
2. Use a window function to compute a rolling 3-day average
3. Extract the result for March 15

## Files
- `solution.sql` → SQL query
- `result.md` → Final numeric result and explanation
